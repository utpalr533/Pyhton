# Pyhton - Python From Code With Harry

This repository contains Python practice files following the **CodeWithHarry** Python tutorial series. It covers fundamental Python concepts for beginners.

## Topics Covered

- Python basics (`01_basics.py`)
- Input/output (`01_input.py`)
- Arithmetic operations (`02 add.py`)
- Data types (`02 data types.py`)
- Comments (`comment.py`)
- Escape sequences (`escape.py`)
- Operators (`operators.py`)
- Tuples (`tupples.py`)
- Match-case statements (`matchcase.py`)
